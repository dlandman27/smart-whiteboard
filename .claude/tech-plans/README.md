# Tech Plans

Tech plans organized by team. Copy `TEMPLATE.md` into the relevant team folder and rename it to describe the feature (e.g., `pet-bar-redesign.md`).

## Folders

| Folder | Team |
| --- | --- |
| `widget-ui/` | Widget UI pod — new widgets, widget redesigns |
| `data-integration/` | Data/Integration pod — new integrations, API changes |
| `backend/` | Backend/Express server changes |
| `frontend/` | Cross-cutting frontend changes (state, routing, etc.) |
| `infrastructure/` | Build, deploy, env, tooling changes |

## Usage

1. Copy `TEMPLATE.md` → `<team>/<feature-name>.md`
2. Fill in all sections
3. Get approvals in the Owners table before starting implementation
